# Trader Performance vs Market Sentiment  
### Hyperliquid | Fear–Greed Analysis

## Overview
This project analyzes how **Bitcoin market sentiment (Fear vs Greed)** relates to **trader behavior and performance** on Hyperliquid.  
The objective is to uncover behavior patterns and derive **actionable strategy rules** that can improve risk-adjusted trading outcomes.

## Methodology
- Daily alignment of sentiment and trade data
- Per-trader daily aggregation (PnL, win rate, leverage, activity)
- Sentiment-based behavioral and performance analysis
- Trader segmentation and lightweight predictive modeling

## How to Run
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```
Open and run:
```bash
notebooks/analysis.ipynb
```

## Key Takeaway
Market sentiment is most effective when used as a **risk-adjustment layer** combined with trader behavior metrics.
